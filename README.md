# 软著源码文档生成助手 (SoftCopyRightDocGen)

## 简介
本软件专为软件著作权申请设计，能够自动扫描项目源码，智能排除依赖库和非代码文件，并生成符合软著申请要求的PDF文档。

## 功能特性
- **智能扫描**：自动排除 `node_modules`, `venv`, `.git` 等无关目录。
- **AI 智能排除**：集成 DeepSeek 等大模型，智能分析项目结构，精准识别并建议排除构建产物、资源文件等非源码内容。
- **注释清除**：支持一键移除代码注释（支持 Python, C/C++, Java, JS 等多种语言），生成更纯净的代码文档。
- **格式规范**：生成的PDF保证每页至少50行代码，统一字体和排版。
- **自动截取**：
    - 若源码总页数 ≤ 60页，输出全部内容。
    - 若源码总页数 > 60页，自动截取前30页和后30页。
- **页眉定制**：自动添加“软件全称 版本号 第X页”页眉。

## 运行环境
- 操作系统：Windows 10/11
- Python版本：3.10+ (推荐 3.13)

## 依赖安装
请在项目根目录下运行：
```bash
pip install -r requirements.txt
```

## 运行方法
请使用您的 Python 解释器运行 `main.py`：
```bash
python main.py
```

## 注意事项
- 默认使用系统自带的“宋体”或“微软雅黑”字体，请确保系统已安装这些字体。
- 建议在生成前先确认项目目录下的文件是否为您需要提交的源码。
